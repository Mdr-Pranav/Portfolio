Ultimate Tic-Tac-Toe

An advanced implementation of Tic-Tac-Toe with AI opponents powered by Reinforcement Learning

Python Flask Q-Learning Reinforcement Learning JavaScript

Project Overview

Ultimate Tic-Tac-Toe is a complex variant of the classic game that adds a strategic layer by including multiple boards. I developed an interactive implementation with a learning AI opponent that improves its gameplay over time through reinforcement learning techniques.

How to Play

  • The game is played on a 3x3 grid of 3x3 sub-boards
  • Players take turns placing their marks (X or O) in empty cells
  • The location of your move determines which sub-board your opponent must play in next
  • Win a sub-board by getting three in a row
  • Win the game by winning three sub-boards in a row

AI Implementation

The AI opponent uses Q-Learning, a type of reinforcement learning algorithm, to improve its gameplay over time:

  • The Q-table is saved after each game, preserving learned knowledge
  • The AI learns from both wins and losses to optimize its strategy
  • The model gradually improves as more games are played
  • Three difficulty levels correspond to different training states of the AI

Technical Features

  • Reinforcement Learning

    Implemented Q-Learning algorithm that allows the AI to learn from experience and improve its gameplay strategy over time through trial and error.

  • Interactive Web Interface

    Created a responsive frontend using HTML, CSS and JavaScript that clearly visualizes the nested board structure and provides an intuitive gaming experience.

  • Flask Backend

    Developed a Flask server that handles game state, processes moves, and integrates with the AI models to provide real-time responses.

  • Difficulty Levels

    Implemented three difficulty settings: Easy (random moves), Medium (partially trained Q-Learning), and Hard (fully trained Q-Learning with optimized parameters).

Project Structure

The codebase is organized into the following key components:

  • app.py: Flask server and API endpoints for game management
  • medium.py: Q-Learning implementation for the AI opponent
  • index.html: Main game interface and user interactions
  • css/styles.css: Styling for the game board and controls
  • js/game.js: Game logic and UI interactions
