Attendance Calculator

A web application to calculate attendance percentage and required attendance for achieving target percentage

HTML CSS JavaScript Python MySQL
Implementation Repository

Project Overview

This dynamic website combines multiple useful tools for students including an Attendance Calculator, CGPA Calculator, and PDF Combiner. The platform is designed to help students manage their academic records efficiently and make quick calculations without the need for manual tracking.

Key Features

  • Attendance Calculator

    Automatically calculates current attendance percentages and predicts how many classes can be missed or should be attended to maintain required attendance levels.

  • CGPA & SGPA Calculator

    Allows students to input course grades and credits to calculate semester GPA and cumulative GPA with precision.

  • PDF Combiner

    Enables merging multiple PDF files into a single document, which is particularly useful for combining assignment files, lecture notes, or presentation slides.

Technical Implementation

The project was developed using modern web technologies:

  • Frontend: HTML5, CSS3, and JavaScript were used to create a responsive and intuitive user interface.
  • Backend: Python was employed for server-side logic, particularly for complex calculations and PDF handling operations.
  • Database: MySQL database stores attendance records, user information, and other relevant data (Currently not used for Onilne Implementation ).
  • Data Storage: Efficient database schema design ensures quick data retrieval and optimal performance.

Challenges & Solutions

Challenge

The self-remembering values in the browser to not input the values again and again

Solution

Implemented a cookie system for CGPA and SGPA pages that will precisely remember the values until you want them to reset.

Challenge

Managing PDF merging with potentially large file sizes.

Solution

Used efficient Python libraries for PDF manipulation and implemented file size limitations with proper user feedback.
